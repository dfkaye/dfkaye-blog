This paragraph is a <p> tag in a content.html file.

Early on I found that creating demos with application code requires inserting raw HTML into the markdown content, using this shortcode by Ana Ulin.

If you don't want to do that, though, you can name the content with the ".html" extension instead, and include the usual front matter at the top of the content file, like so: 


---
title: "hugo using html content demo"
date: 2020-07-08T13:40:43-07:00
draft: false
---

<p>
  This paragraph is a <code><p></code> tag in a content.html file.
</p>
